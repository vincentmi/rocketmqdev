# rocketmqdev

RocketMQ 镜像包含nameserver broker 和rocketmq-console 主要用于开发环境调试.
**请勿用于生产环境**

## 端口映射
| 端口 | 服务 |
| --- | --- |
| 9876 | rocket nameserver|
| 8080 | rocketmq-console |
